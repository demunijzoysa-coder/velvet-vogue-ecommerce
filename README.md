# Velvet Vogue – Luxury Fashion E-Commerce Website

Velvet Vogue is a responsive, multi-page e-commerce website developed as part of the **HND in Web Design & Development**. The project simulates a modern online fashion retail platform, demonstrating front-end development principles, user interaction design, accessibility considerations, and basic e-commerce functionality without the use of server-side technologies.

The website focuses on clean UI design, realistic user workflows, and professional development practices aligned with real-world web development standards.

---

## Project Objectives

The primary objectives of this project are to:

- Design and develop a branded, multi-page e-commerce website
- Demonstrate product browsing, filtering, and cart functionality
- Implement simulated user authentication and account management
- Provide an admin interface for product management (simulation)
- Apply responsive design, accessibility principles, and usability standards
- Showcase a structured development and testing process

---

## Technologies Used

- **HTML5** – Semantic structure and page layout  
- **CSS3** – Styling, responsive layouts, and visual branding  
- **Vanilla JavaScript** – Client-side logic and interactivity  
- **LocalStorage** – Simulated persistence for cart, users, and admin data  
- **GitHub** – Version control and project management  

No external frameworks or backend services are used.

---

## Website Features

### User-Facing Features
- Home page with hero section and featured products
- Product listing with category, size, and price filters
- Individual product detail pages
- Shopping cart with quantity updates and removal
- Checkout simulation (no real payment processing)
- User login and registration simulation
- User account dashboard with order history
- Contact / inquiry form with validation

### Admin Features (Simulation)
- Role-based admin access
- Product management dashboard
- Add and delete products
- Search and product listing table
- LocalStorage-based data persistence
